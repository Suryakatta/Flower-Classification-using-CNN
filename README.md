# Flower Classification using CNN

This project uses Convolutional Neural Networks (CNN) with TensorFlow and Keras to classify flower images.

## Features
- CNN Architecture
- Data Augmentation
- Dropout Regularization
- Softmax Classification
- Image Prediction
- TensorFlow/Keras

## Technologies Used
- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib

## Model Architecture
- Conv2D
- MaxPooling2D
- Dropout
- Dense Layers
- Softmax Output

## Accuracy
- Training Accuracy: 88%
- Test Accuracy: 74%

## Dataset
TensorFlow Flower Dataset
